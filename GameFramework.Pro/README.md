# Overview

The framework that allows you to design high-quality architecture of your game project.

# Reference

The framework consists of the following classes:

- ProgramBase
- UI
    - ThemeBase
    - ScreenBase
    - WidgetBase
    - ViewableWidgetBase
    - ViewBase
    - RouterBase
- App
    - ApplicationBase
- Domain (Business)
    - GameBase
    - PlayerBase
    - EntityBase
